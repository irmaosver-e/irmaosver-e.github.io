    Why Skitchin'?

    Looking for a new project to tackle, I remembered a comment on my Eliminate Down Remix video from marcos-lq2nh, who mentioned how awkward it felt to switch weapons in Skitchin’ and suggested adding 6-button controller support.
    When I replayed the game to see what could be improved, the first thing that bothered me was the password system. The passwords are so long that I often didn’t feel like writing them down or entering them again to continue, especially since the game generates new ones so frequently. So I felt the game would benefit greatly from a proper save system.
    Thanks to marcos-lq2nh for the suggestion!

    Hack features:

    • SRAM save system with three available slots, Load or deleting save files is done from the password screen. Saving progress can be done before each race.
    • SRAM Password Saving: After clearing a stage, the last password you received is automatically saved to SRAM.
    • Region Lock Disabled
