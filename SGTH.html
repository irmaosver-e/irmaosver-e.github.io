To play this hack, you’ll need the patch provided and the original ROM, which you'll have to source yourself.
You can apply the patch using this online tool: https://www.romhacking.net/patch/

Why “Spot Goes to Hollywood”?

This hack was requested by the YouTube user marcos-lq2nh who commented on my Desert Strike hack video, pointing out how the original control scheme made the gameplay frustrating.
Rather than simply replacing the controls, I thought it would be better to add the new scheme as an additional option in the controls menu.
Thanks to marcos-lq2nh for the suggestion!

Hack features:

  • two control styles from the Options menu:
    - Original: Classic isometric movement (up/down/left/right = diagonal directions).
    - Revamped: Direct movement based on button input (up moves up, left moves left, etc.).
  • SRAM Password Saving: After clearing a stage, the last password you received is automatically saved to SRAM.
  • Region Lock Disabled
